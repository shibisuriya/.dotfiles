## Default keybindings

Note: An item could be a file or a directory.

- Use the 'Space' key to select or unselect an item.

- Use the 'v' key to invert the selection (the items that
  were unselected will be selected & the items that were
  selected will be unselected).

- Use the 'u' key to unselect all the selected items.

- Use the 'y' key to copy an item.

- Use the 'c' key to clear the items that were copied or
  cut.

- Use the 'p' key to paste the items that were cut or
  copied.

## Keybindings that I added

- Use the 'x' to cut an item.
